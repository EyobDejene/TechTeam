## Installation

Clone the repo
```bash
git clone [projectTech]
```

After the cloning the repo
```bash
cd [projectTech]
```

With [npm](https://www.npmjs.com/) installed, run in the root directory
```bash
npm install
```

## Usage
Start the server
```bash
npm start 
```
The application runs at 127.0.0.1 on port 3000
